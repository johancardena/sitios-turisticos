# Aplicación de Sitios Turísticos en Flutter

## Descripción

Esta aplicación móvil fue desarrollada utilizando Flutter y permite visualizar información de 10 sitios turísticos representativos del Ecuador.
La aplicación presenta información detallada de cada lugar turístico, incluyendo nombre, ubicación, imagen, descripción, coordenadas geográficas y la posibilidad de marcar sitios favoritos.
Además, incorpora integración con Google Maps para visualizar la ubicación exacta de cada destino turístico.

## Características

* Visualización de 10 sitios turísticos.
* Imágenes almacenadas localmente.
* Descripción de cada lugar.
* Coordenadas geográficas (latitud y longitud).
* Selección de favoritos mediante ícono de corazón.
* Apertura de ubicación en Google Maps.
* Interfaz desarrollada con Material Design.

## Tecnologías Utilizadas

* Flutter
* Dart
* Material Design
* url_launcher

### Favoritos

Los usuarios pueden marcar y desmarcar lugares turísticos utilizando el ícono de corazón.

### Geolocalización

Cada sitio turístico muestra su latitud y longitud.

### Integración con Google Maps

Mediante el botón "Ver en Google Maps", el usuario puede abrir directamente la ubicación correspondiente en Google Maps.
